# Aquarium Management API

REST API for managing aquarium inventory, species, and maintenance records.

## Tech Stack

- **Runtime:** Node.js
- **Framework:** Express
- **Database:** MySQL
- **Authentication:** JWT
- **Security:** bcrypt

## Features (Planned)

- 🐠 Species and tank management
- 📦 Inventory tracking
- 🔐 JWT authentication with role-based access
- 📋 Maintenance records

## Installation

```bash
# Clone repository
git clone https://github.com/Mateoquil/Aquarium-Management-API.git
cd Aquarium-Management-API

# Install dependencies
npm install

# Configure environment variables
cp .env.example .env

# Run
npm run dev
```
